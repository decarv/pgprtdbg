# Introduction

[**pgprtdbg**][pgprtdbg] is an application that turns [PostgreSQL](https://www.postgresql.org) protocol v3 interaction into a trace file.

The supported platforms are

* [Fedora][fedora] 32+
* [RHEL][rhel] 8 / RockyLinux 8
* [RHEL][rhel] 9 / RockyLinux 9
* [FreeBSD][freebsd]
* [OpenBSD][openbsd]
